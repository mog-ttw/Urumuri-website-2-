# Urumuri Delivery Website

A modern and responsive landing page for **Urumuri**, a delivery and courier service platform operating in Kigali and nearby districts in Rwanda.

The website showcases delivery services, app promotion, customer engagement features, and animated interactive UI elements built with pure HTML, CSS, and JavaScript.

---

##  Features

- Responsive modern UI
- Smooth scrolling navigation
- Animated hero sections
- Interactive buttons with ripple effects
- Scroll-triggered animations
- Delivery service showcase
- App promotion section
- Animated statistics counters
- Mobile-friendly navigation menu
- Social media footer
- 3D hover effects
- Parallax bike rider background animations
